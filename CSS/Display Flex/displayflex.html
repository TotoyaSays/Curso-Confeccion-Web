justify-content Distribuir los elementos horizontales
  • flex-start: Alinea los elementos a la izquierda del contenedor
  • flex-end: Alinea los elementos a la derecha del contenedor
  • center: Alinea los elementos al centro del contenedor
  • space-between: Alinea los elementos dejando el mismo espacio entre ellos
  • space-around: Alinea los elementos dejando la misma separacion alrededor de ellos
align-items Distribuir los elementos verticalmente
  • flex-start: Alinea elementos a la parte superior del contenedor
  • flex-end: Alinea elementos a la parte inferior del contenedor
  • center: Alinea elementos en el centro (verticalmente hablando) del contenedor
  • baseline: Muestra elementos en la línea base del contenedor
  • stretch: Elementos se estiran para ajustarse al contenedor
flex-direction Define la dirección de los elementos en el contenedor
  • row: Elementos son colocados en la misma dirección del texto.
  • row-reverse: Elementos son colocados en la dirección opuesta al texto.
  • column: Elementos se colocan de arriba hacia abajo.
  • column-reverse: Elementos se colocan de abajo hacia arriba.
flex-wrap Permite o no colocar elementos en filas adicionales
  • nowrap: Fuerza a que todos los elemntos esten en la misma linea.
  • wrap: Los elemntos puede pueden ocupar lineas adicionales
  • wrap-reverse: como wrap pero como en un espejo
flex-flow Es flex-direction+flex-wrap Ponemos los valores de flex-direction + flex-wrap
order Cambia el orden de UN elemento
  • Un numero positivo de posiciones hacia la derecha o un numero negativo de posiciones hacia la izquierda
align-self Lo mismo que align-items pero SOLO con UN elemento
  • Acepta los mismos valores que align-items
align-content Define la dirección de los elementos en el contenedor
  • flex-star: Las filas se colocan arriba
  • flex-end: Las filas se colocan abajo
  • center: Las filas se colocan centradas
  • space-between: Las filas se colocan separadas con la misma distancia entre elllas
  • space-around: Las filas se colocan con las misma separacion entre ellas
  • stretch: Las filas se estiran para ajustarse al contenedor
