AQUI VA EL TITULO AQUI IRIA UNA AGRUPACION
El valor del atriuto for tiene que coincidir con el valor del atributo ID del input
  • name: Cuando viaja por la URL va bajo este nombre. ?name=valorQueEscriba
  • value:Es el valor que escribe el usuario y que viajara por la URL
  • size: Especifica el ancho en numero de caracteres
required: Se pone solo. Si se pone debe rellenarse para que se pueda enviar el formulario.
  • checked: Si está escrito la casilla aparece marcada.
  • name: Todos tienen el mismo "name" y si se marca uno los otros se desmarcan
