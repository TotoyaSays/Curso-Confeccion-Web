Espacio div p {} Pone las propiedades a los p que estan dentro del div
* * {}, div * Selecciona TODO, o todo lo que hay dentro de lo que le digas
, h1,p,a Aplica a todos, funciona como la coma normal
. .class Aplica el formato a el nombre de la clase
# #identificador Aplica el formato a el nombre de la id
+ div+p Aplica el formato a la siguiente cosa que sea hermana no a lo que tenga dentro
> div>p Aplica el formato a algo que este dentro de una cosa
~ div~p Aplica el formato a lo que esta justo detras de lo que le digas
